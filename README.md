# SangerCleanup
## Summary
The purpose of the *Sanger Cleanup* repository is to serve as the home for the SangerCleanup_Processes sub-method, SangerCleanup_Template method and their respective compenents.
The aim of cleanup redesign is to better align current methods with standardisation efforts and future automation strategy. 
One possibility is the use of a sub-method library to perform steps of the cleanup, allowing users to chain blocks of sub-methods from a central repository into a single functional cleanup method.


The new methods will reduce the amount of layouts required for each method by consolidating similar methods into one generic layout (where possible). 
Methods will also have functionality built-in to select between one and two plate variations, rather than having this be two separate methods.
